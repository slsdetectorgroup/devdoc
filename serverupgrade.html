Detector Server Upgrade¶

Location: slsDetectorPackage/serverBin/ folder for every release.

  1. Install tftp and copy detector server binary to tftp folder

  2. Program from console

    Note

    These instructions are for upgrades from v5.0.0. For earlier versions, contact us.

    # copies new server from pc tftp folder, creates a soft link to xxxDetectorServerxxx
# [Jungfrau][CTB][Moench] also edits initttab to respawn server on reboot
# Then, the detector controller will reboot (except Eiger)
sls_detector_put copydetectorserver xxxDetectorServerxxx pcxxx

  3. Copy the detector server specific config files or any others required to the detector:

    sls_detector_put execcommand "tftp pcxxx -r configxxx -g"

Note

For Mythen3, Gotthard2 and Eiger, you need to add scripts to automatically start detector server upon power on. See Automatic start for more details.

Note

Eiger requires a manual reboot. Or killall the servers and restart the new linked one. If you are in the process of updating firmware, then don’t reboot yet.

Errors¶

  1. tftp write error: There is no space left. Please delete some old binaries and try again.

  2. text file busy: You are trying to copy the same server.
