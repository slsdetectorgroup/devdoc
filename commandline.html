Command line interface¶

Usage¶

Commands can be uses either with sls_detector_get or sls_detector_put

sls_detector_get vrf

Commands¶

acquire
Acquire the number of frames set up.
activate [0, 1] [(optional) padding|nopadding]
[Eiger] 1 is default. 0 deactivates readout and does not send data.
Padding will pad data files for deactivates readouts.
adc [n_channel (0-7 for channel|8 for temperature)]
[Ctb] Slow ADC channel in mV or °C.
adcclk [n_clk in MHz]
[Ctb][Moench] ADC clock frequency in MHz.
adcenable [bitmask]
[Ctb][Moench] ADC Enable Mask for 1Gb Mode for each 32 ADC channel.
adcenable10g [bitmask]
[Ctb][Moench] ADC Enable Mask for 10Gb mode for each 32 ADC channel. However, if any of consecutive 4 bits are enabled, the complete 4 bits are enabled.
adcinvert [bitmask]
[Ctb][Moench][Jungfrau] ADC Inversion Mask.
[Jungfrau][Moench] Inversions on top of the default mask.
adcphase [n_value] [(optional)deg]
[Jungfrau][Ctb][Moench][Gotthard] Phase shift of ADC clock.
[Jungfrau] Absolute phase shift. If deg used, then shift in degrees. Changing Speed also resets adcphase to recommended defaults.
[Ctb][Moench] Absolute phase shift. If deg used, then shift in degrees. Changing adcclk also resets adcphase and sets it to previous values.
[Gotthard] Relative phase shift. Cannot get
adcpipeline [n_value]
[Ctb][Moench] Pipeline for ADC clock.
adcreg [address] [value]
[Jungfrau][Ctb][Moench][Gotthard] Writes to an adc register in hex. Advanced user Function!
adcvpp [dac or mV value][(optional unit) mV]
[Ctb][Moench] Vpp of ADC.
0 -> 1V ; 1 -> 1.14V ; 2 -> 1.33V ; 3 -> 1.6V ; 4 -> 2V.
asamples [n_samples]
[CTB][Moench] Number of analog samples expected.
auto_comp_disable [0, 1]
[Jungfrau] Auto comparator disable mode. By default, the on-chip gain switching is active during the entire exposure.This mode disables the on - chip gain switching comparator automatically after 93.75% of exposure time (only for longer than 100us).
Default is 0 or this mode disabled(comparator enabled throughout). 1 enables mode. 0 disables mode.
badchannels [fname]
[Gotthard2] Sets the bad channels (from file of bad channel numbers) to be masked out.
burstmode [off or 0, internal or 1, external or 2]
[Gotthard2] Default is burst internal type
burstperiod [duration] [(optional unit) ns|us|ms|s]
[Gotthard2] Burst period. Only in burst mode and auto timing mode.
bursts [n_bursts]
[Gotthard2] Number of bursts per aquire. Only in auto timing mode and burst mode. Use timing command to set timing mode and burstmode command to set burst mode.
bustest
[Jungfrau][Gotthard][Mythen3][Gotthard2][Ctb][Moench] Bus test, ie. keeps writing and reading back different values in R/W register.
cdsgain [0, 1]
[Gotthard2] Enable or disable CDS gain. Default is disabled.
clearbit [n_value]
[Moench] Maximum energy threshold (soft setting) for processor.
clearbusy
Clears Acquiring Flag for unexpected acquire command terminations.
clearroi
[Gotthard] Resets Region of interest in detector. All channels enabled. Default is all channels.
clientversion
Client software version in format [0xYYMMDD].
clkdiv [n_clock (0-8)] [n_divider]
[Gotthard2][Mythen3] Clock Divider of clock n_clock. Must be greater than 1.
clkfreq [n_clock (0-8)] [freq_in_Hz]
[Gotthard2][Mythen3] Frequency of clock n_clock in Hz. Use clkdiv to set frequency.
clkphase [n_clock (0-8)] [phase] [deg (optional)]
[Gotthard2][Mythen3] Phase of clock n_clock. If deg, then phase shift in degrees, else absolute phase shift values.
confadc [chip index 0-10, -1 for all] [adc index 0-31, -1 for all] [12 bit configuration value in hex]
[Gotthard2] Sets configuration for specific chip and adc, but configures 1 chip (all adcs for that chip) at a time.
config [fname]
Frees shared memory before loading configuration file. Set up once.
copydetectorserver [server_name] [pc_host_name]
[Jungfrau][Ctb][Moench][Mythen3][Gotthard2] Copies detector server via tftp from pc and changes respawn server name in /etc/inittab of detector.
counters [i0] [i1] [i2]…
[Mythen3] List of counters indices enabled. Each element in list can be 0 - 2 and must be non repetitive.
currentsource [0, 1]
[Gotthard2] Enable or disable current source. Default is disabled.
dac [dac index] [dac or mV value] [(optional unit) mV]
[Ctb] Dac.
daclist
Gets the list of commands for every dac for this detector.
dacvalues [(optional unit) mV]
Gets the values for every dac for this detector.
dbitclk [n_clk in MHz]
[Ctb] Clock for latching the digital bits in MHz.
dbitphase [n_value] [(optional)deg]
[Ctb][Jungfrau] Phase shift of clock to latch digital bits. Absolute phase shift. If deg used, then shift in degrees.
[Ctb]Changing dbitclk also resets dbitphase and sets to previous values.
dbitpipeline [n_value]
[Ctb] Pipeline of the clock for latching digital bits.
delay [duration] [(optional unit) ns|us|ms|s]
[Jungfrau][Gotthard][Mythen3][Gotthard2][Ctb][Moench] Delay after trigger
delayl
[Gotthard][Jungfrau][Mythen3][Gotthard2][CTB][Moench] DelayLeft Delay Left in Acquisition.
[Gotthard2] only in continuous mode.
detectormode [counting|interpolating|analog]
[Moench] Detector mode (soft setting) in processor.
detectornumber
Receiver version in format [0xYYMMDD].
detectorserverversion
On-board detector server software version in format [0xYYMMDD].
detsize [nx] [ny]
Detector size, ie. Number of channels in x and y dim. If 0, then hostname adds all modules in y dim. This is used to calculate module coordinates included in UDP data packet header.
diodelay [64 bit bitmask] [0-775]
[Ctb] Delay for digital IO pins selected by the bitmask. Delay is in ps and max of 775 ps. Resolution is 25 ps.
dr [value]
Dynamic Range or number of bits per pixel in detector.
[Eiger] Options: 4, 8, 16, 32
[Mythen3] Options: 8, 16, 32
[Jungfrau][Gotthard][Ctb][Moench][Mythen3][Gotthard2] 16
drlist
Gets the list of dynamic ranges for this detector.
dsamples [n_value]
[CTB] Number of digital samples expected.
emax [n_value]
[Moench] Maximum energy threshold (soft setting) for processor.
emin [n_value]
[Moench] Minimum energy threshold (soft setting) for processor.
execcommand [command]
Executes command on detector server.
exptime [duration] [(optional unit) ns|us|ms|s]
[Eiger][Jungfrau][Gotthard][Gotthard2][Moench][Ctb] Exposure time
[Mythen3] Exposure time of all gate signals in auto and trigger mode (internal gating). To specify gate index, use exptime1, exptime2, exptime3.
exptime1 [n_value]
[Mythen3] Exposure time of gate signal 1 in auto and trigger mode (internal gating).
exptime2 [n_value]
[Mythen3] Exposure time of gate signal 2 in auto and trigger mode (internal gating).
exptime3 [n_value]
[Mythen3] Exposure time of gate signal 3 in auto and trigger mode (internal gating).
exptimel [(optional unit) ns|us|ms|s]
[Gotthard] Exposure time left for current frame.
extsampling [0, 1]
[Ctb] Enable for external sampling signal to extsamplingsrc signal for digital data. For advanced users only.
extsamplingsrc [0-63]
[Ctb] Sampling source signal for digital data. For advanced users only.
extsig [n_signal] [signal_type] External signal mode for trigger timing mode.
[Gotthard] [0] [trigger_in_rising_edge|trigger_in_falling_edge]
[Mythen3] [0-7] [trigger_in_rising_edge|trigger_in_falling_edge|inversion_on|inversion_off]
where 0 is master input trigger signal, 1-3 is master input gate signals, 4 is busy out signal and 5-7 is master output gate signals
fformat [binary|hdf5]
File format of data file. For HDF5, package must be compiled with HDF5 flags. Default is binary.
filter [0|1|2|3]
[Gotthard2] Set filter resistor. Default is 0.
findex [n_value]
File or Acquisition index.
firmwaretest
[Jungfrau][Gotthard][Mythen3][Gotthard2][Ctb][Moench] Firmware test, ie. reads a read fixed pattern from a register.
firmwareversion
Fimware version of detector in format [0xYYMMDD] or integer for Eiger.
flippeddatax [0, 1]
[Eiger] Top or Bottom Half of Eiger module. 1 is bottom, 0 is top. Used to let Receivers and Gui know to flip the bottom image over the x axis. Files are not written without the flip however.
flowcontrol10g [0, 1]
[Eiger][Jungfrau] 10GbE Flow Control.
fmaster [0, 1]
Enable or disable receiver master file. Default is 1.
fname [name]
File name prefix for output data file. Default is run. File name: [file name prefix]_d[detector index]_f[sub file index]_[acquisition/file index].raw.
foverwrite [0, 1]
Enable or disable file overwriting. Default is 1.
fpath [path]
Directory where output data files are written in receiver. If path does not exist, it will try to create it.
framemode [pedestal|newpedestal|flatfield|newflatfield]
[Moench] Frame mode (soft setting) in processor.
frames [n_frames]
Number of frames per acquisition. In trigger mode, number of frames per trigger. Cannot be set in modular level. In scan mode, number of frames is set to number of steps.
[Gotthard2] Burst mode has a maximum of 2720 frames.
framesl
[Gotthard][Jungfrau][Mythen3][Gotthard2][CTB][Moench] Number of frames left in acquisition.
[Gotthard2] only in continuous mode.
free
Free detector shared memory
fwrite [0, 1]
Enable or disable receiver file write. Default is 1.
gappixels [0, 1]
[Eiger][Jungfrau] Include Gap pixels in client data call back in Detecor api. Will not be in detector streaming, receiver file or streaming. Default is 0.
gatedelay [duration] [(optional unit) ns|us|ms|s]
[Mythen3] Gate Delay of all gate signals in auto and trigger mode (internal gating).
gatedelay1 [n_value]
[Mythen3] Gate Delay of gate signal 1 in auto and trigger mode (internal gating).
gatedelay2 [n_value]
[Mythen3] Gate Delay of gate signal 2 in auto and trigger mode (internal gating).
gatedelay3 [n_value]
[Mythen3] Gate Delay of gate signal 3 in auto and trigger mode (internal gating).
gates [n_gates]
[Mythen3] Number of external gates in gating or trigger_gating mode (external gating).
getbit [n_value]
[Moench] Maximum energy threshold (soft setting) for processor.
highvoltage [n_value]
High voltage to the sensor in Voltage.
[Gotthard] [0|90|110|120|150|180|200]
[Eiger][Mythen3][Gotthard2] 0-200
[Jungfrau][Ctb][Moench] [0|60-200]

Frees shared memory and sets hostname (or IP address) of all modules concatenated by +.

[Gotthard] 1 adds channel intensity with precalculated values when taking an acquisition. Default is 0.
[Eiger][Jungfrau] Only for Virtual servers. If 0, each pixel intensity incremented by 1. If 1, all pixels almost saturated.

Enable or disable intial compatibility and other checks at detector start up. It is enabled by default. Must come before ‘hostname’ command to take effect. Can be used to reprogram fpga when current firmware is incompatible.

[Gotthard2] Inject channels with current source for calibration. Offset is starting channel that is injected, increment determines succeeding channels to be injected.

rx_udpsocksize [n_size]
UDP socket buffer size in receiver. Tune rmem_default and rmem_max accordingly.
rx_version
Receiver version in format [0xYYMMDD].

rx_zmqport [port]
Zmq port for data to be streamed out of the receiver. Also restarts receiver zmq streaming if enabled. Default is 30001. Modified only when using an intermediate process between receiver and client(gui). Must be different for every detector (and udp port). Multi command will automatically increment for individual modules.
rx_zmqstartfnum [fnum]
The starting frame index to stream out. 0 by default, which streams the first frame in an acquisition, and then depending on the rx zmq frequency/ timer
rxb_lb [dac or mV value][(optional unit) mV]
