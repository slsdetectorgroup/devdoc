Command line interface¶

Usage¶

Commands can be uses either with sls_detector_get or sls_detector_put

sls_detector_get vrf

Commands¶

acquire - Acquire the number of frames set up.
activate [0, 1] [(optional) padding|nopadding]

[Eiger] 1 is default. 0 deactivates readout and does not send data. Padding will pad data files for deactivates readouts.

adc [n_channel (0-7 for channel|8 for temperature)]

[Ctb] Slow ADC channel in mV or °C.

adcclk [n_clk in MHz]

[Ctb][Moench] ADC clock frequency in MHz.

adcenable [bitmask]

[Ctb][Moench] ADC Enable Mask for 1Gb Mode for each 32 ADC channel.

adcenable10g [bitmask]

[Ctb][Moench] ADC Enable Mask for 10Gb mode for each 32 ADC channel. However, if any of consecutive 4 bits are enabled, the complete 4 bits are enabled.

adcinvert [bitmask]

[Ctb][Moench][Jungfrau][Moench] ADC Inversion Mask. [Jungfrau][Moench] Inversions on top of the default mask.

adcphase [n_value] [(optional)deg]

[Jungfrau][Ctb][Moench][Moench][Gotthard] Phase shift of ADC clock. [Jungfrau] Absolute phase shift. If deg used, then shift in degrees. Changing Speed also resets adcphase to recommended defaults. [Ctb][Moench] Absolute phase shift. If deg used, then shift in degrees. Changing adcclk also resets adcphase and sets it to previous values. [Gotthard] Relative phase shift

adcpipeline [n_value]

[Ctb][Moench] Pipeline for ADC clock.

adcreg [address] [value]

[Jungfrau][Ctb][Moench][Gotthard] Writes to an adc register in hex.

adcvpp [dac or mv value][(optional unit) mv]
[Ctb][Moench] Vpp of ADC.

0 -> 1V ; 1 -> 1.14V ; 2 -> 1.33V ; 3 -> 1.6V ; 4 -> 2V.

asamples [0, 1]

[CTB][Moench] Number of analog samples expected.

auto_comp_disable [0, 1]

[Jungfrau] Auto comparator disable mode. Default 0 or this mode disabled(comparator enabled throughout). 1 enables mode. 0 disables mode. This mode disables the on-chip gain switching comparator automatically after 93.75% of exposure time (only for longer than 100us).

burstmode [off or 0, internal or 1, external or 2]

[Gotthard2] Default is burst internal type

burstperiod [duration] [(optional unit) ns|us|ms|s]

[Gotthard2] Burst period. Only in burst mode and auto timing mode.

bursts [n_bursts]

[Gotthard2] Number of bursts per aquire. Only in auto timing mode and burst mode. Use timing command to set timing mode and burstmode command to set burst mode.

bustest

[Jungfrau][Gotthard][Mythen3][Gotthard2][Ctb][Moench] Bus test, ie. keeps writing and reading back different values in R/W register.

clearbit [n_value]

[Moench] Maximum energy threshold (soft setting) for processor.

clearbusy

Clears Acquiring Flag for unexpected acquire command terminations.

clearroi

[Gotthard] Resets Region of interest in detector. All channels enabled. Default is all channels.

clientversion

Client software version in format [0xYYMMDD].

clkdiv [n_clock (0-8)] [n_divider]

[Gotthard2][Mythen3] Clock Divider of clock n_clock. Must be greater than 1.

clkfreq [n_clock (0-8)] [freq_in_Hz]

[Gotthard2][Mythen3] Frequency of clock n_clock in Hz. Use clkdiv to set frequency.

clkphase [n_clock (0-8)] [phase] [deg (optional)]

[Gotthard2][Mythen3] Phase of clock n_clock. If deg, then phase shift in degrees, else absolute phase shift values.

config [fname]

Configures detector to configuration contained in fname. Set up once.

copydetectorserver [server_name] [pc_host_name]

[Jungfrau][Ctb][Moench] Copies detector server via tftp from pc and changes respawn server name in /etc/inittab of detector.

counters [i0] [i1] [i2]…

[Mythen3] List of counters enabled. Each element in list can be 0 - 2 and must be non repetitive.

currentsource [0, 1]

[Gotthard2] Enable or disable current source. Default is disabled.

dac [dac index] [dac or mv value] [(optional unit) mv]

[Ctb] Dac.

daclist

Gets the list of commands for every dac for this detector.

dacvalues

Gets the list of commands for every dac for this detector.

dbitclk [n_clk in MHz]

[Ctb] Clock for latching the digital bits in MHz.

dbitphase [n_value] [(optional)deg]

[Ctb][Jungfrau] Phase shift of clock to latch digital bits. Absolute phase shift. If deg used, then shift in degrees. [Ctb]Changing dbitclk also resets dbitphase and sets to previous values.

dbitpipeline [n_value]

[Ctb] Pipeline of the clock for latching digital bits.

delay [duration] [(optional unit) ns|us|ms|s]

[Jungfrau][Gotthard][Mythen3][Gotthard2][Ctb][Moench] Delay after trigger

delayl

[Gotthard][Jungfrau][Mythen3][Gotthard2][CTB][Moench] DelayLeft Delay Left in Acquisition. [Gotthard2] only in continuous mode.

detectormode [counting|interpolating|analog]

[Moench] Detector mode (soft setting) in processor.

detectornumber

Receiver version in format [0xYYMMDD].

detectorserverversion

On-board detector server software version in format [0xYYMMDD].

detsize [nx] [ny]

Detector size, ie. Number of channels in x and y dim. If 0, then hostname adds all modules in y dim. This is used to calculate module coordinates included in UDP data packet header.

diodelay [64 bit bitmask] [0-775]

[Ctb] Delay for digital IO pins selected by the bitmask. Delay is in ps and max of 775 ps. Resolution is 25 ps.

dr [4|8|16|32]

[Eiger] Dynamic Range or number of bits per pixel in detector.

dsamples [0, 1]

[CTB] Number of digital samples expected.

emax [n_value]

[Moench] Maximum energy threshold (soft setting) for processor.

emin [n_value]

[Moench] Minimum energy threshold (soft setting) for processor.

execcommand [command]

Executes command on detector server.

exptime [duration] [(optional unit) ns|us|ms|s]

Exposure time [Gotthard2] Uploaded to detector just before acquisition starts

exptimel [(optional unit) ns|us|ms|s]

[Gotthard] Exposure time left for current frame.

extsampling [0, 1]

[Ctb] Enable for external sampling signal to extsamplingsrc signal for digital data. For advanced users only.

extsamplingsrc [0-63]

[Ctb] Sampling source signal for digital data. For advanced users only.

extsig [trigger_in_rising_edge|trigger_in_falling_edge]

[Gotthard] External signal mode for trigger timing mode.

fformat [binary|hdf5]

File format of data file. For HDF5, package must be compiled with HDF5 flags. Default is binary.

findex [0, 1]

File or Acquisition index.

firmwaretest

[Jungfrau][Gotthard][Mythen3][Gotthard2][Ctb][Moench] Firmware test, ie. reads a read fixed pattern from a register.

firmwareversion

Fimware version of detector in format [0xYYMMDD] or integer for Eiger.

flippeddatax [0, 1]

[Eiger] Top or Bottom Half of Eiger module. 1 is bottom, 0 is top. Used to let Receiver and Gui know to flip the bottom image over the x axis.

flowcontrol10g [0, 1]

[Eiger][Jungfrau] 10GbE Flow Control.

fmaster [0, 1]

Enable or disable receiver master file. Default is 1.

fname [path]

File name prefix for output data file. Default is run. File name: [file name prefix]_d[detector index]_f[sub file index]_[acquisition/file index].raw.

foverwrite [0, 1]

Enable or disable file overwriting. Default is 1.

fpath [path]

Directory where output data files are written in receiver.

framemode [pedestal|newpedestal|flatfield|newflatfield]

[Moench] Frame mode (soft setting) in processor.

frames [n_frames]

Number of frames per aquire. In trigger mode, number of frames per trigger. [Gotthard2] Burst mode has a maximum of 2720 frames.

framesl

[Gotthard][Jungfrau][Mythen3][Gotthard2][CTB][Moench] Number of frames left in acquisition. [Gotthard2] only in continuous mode.

freeee detector shared memory
fwrite [0, 1]

Enable or disable receiver file write. Default is 1.

gappixels [0, 1]

[Eiger] Include Gap pixels in data file or data call back. 4 bit mode gap pixels only ind ata call back.

getbit [n_value]

[Moench] Maximum energy threshold (soft setting) for processor.

hostname

Frees shared memory and sets hostname (or IP address) of all modules concatenated by +.

ib_test_c [dac or mv value][(optional unit) mv]

[Gotthard] Dac for ??

ibias_sfp [dac or mv value][(optional unit) mv]

[Moench] Dac for 7

im_a

[Ctb] Measured current of power supply a in mA.

im_b

[Ctb] Measured current of power supply b in mA.

im_c

[Ctb] Measured current of power supply c in mA.

im_d

[Ctb] Measured current of power supply d in mA.

im_io

[Ctb] Measured current of power supply io in mA.

imagetest [0, 1]

[Gotthard] 1 adds channel intensity with precalculated values when taking an acquisition. Default is 0.

initialchecks [0, 1]

Enable or disable intial compatibility and other checks at detector start up. It is enabled by default. Must come before ‘hostname’ command to take effect. Can be used to reprogram fpga when current firmware is incompatible.

inj_ch [offset] [increment]

[Gotthard2] Inject channels with current source for calibration. Offset is starting channel that is injected, increment determines succeeding channels to be injected.

interruptsubframe [0, 1]

[Eiger] 1 interrupts last subframe at required exposure time. 0 will wait for last sub frame to finish exposing. 0 is default.

iodelay [dac or mv value][(optional unit) mv]

[Eiger] Dac for ??

lastclient

Client IP Address that last communicated with the detector.

led [0, 1]

[Ctb] Switches on/off all LEDs.

list

lists all available commands, list deprecated - list deprecated commands

lock [0, 1]

Lock detector to one IP, 1: locks

maxadcphaseshift

[Jungfrau][CTB][Moench] Absolute maximum Phase shift of ADC clock.

maxclkphaseshift [n_clock (0-8)]

[Gotthard2][Mythen3] Absolute Maximum Phase shift of clock n_clock.

maxdbitphaseshift

[CTB][Jungfrau] Absolute maximum Phase shift of of the clock to latch digital bits.

measuredperiod [(optional unit) ns|us|ms|s]

[Eiger] Measured frame period between last frame and previous one. Useful data only for acquisitions with more than 1 frame.

measuredsubperiod [(optional unit) ns|us|ms|s]

[Eiger] Measured sub frame period between last sub frame and previous one.

nframes

[Jungfrau][Mythen3][Gotthard2][Moench][CTB][Moench] Number of frames from start run control. [Gotthard2] only in continuous mode.

now [(optional unit) ns|us|ms|s]

[Jungfrau][Mythen3][Gotthard2][Moench][CTB] Time from detector start up. [Gotthard2] only in continuous mode.

numinterfaces [1, 2]

[Jungfrau] Number of udp interfaces to stream data from detector. Default: 1.

overflow [0, 1]

[Eiger] Enable or disable show overflow flag in 32 bit mode.

Package version (git branch).

parallel [0, 1]

[Eiger] Enable or disable parallel mode.

parameters [fname]

Sets detector measurement parameters to those contained in fname. Set up per measurement.

partialreset [0, 1]

[Eiger] Sets up detector to do partial or complete reset at start of acquisition. 0 complete reset, 1 partial reset.

patclkctrl [64 bit mask]

[Ctb][Moench][Mythen3] 64 bit mask defining output clock enable.

patioctrl [64 bit mask]

[Ctb][Moench][Mythen3] 64 bit mask defining input (0) and output (1) signals.

patlimits [start addr] [stop addr]

[Ctb][Moench][Mythen3] Limits of complete pattern.

patloop0 [start addr] [stop addr]

[Ctb][Moench][Mythen3] Limits of loop 0.

patloop1 [start addr] [stop addr]

[Ctb][Moench][Mythen3] Limits of loop 1.

patloop2 [start addr] [stop addr]

[Ctb][Moench][Mythen3] Limits of loop 2.

patmask [64 bit mask]

[Ctb][Moench][Mythen3] 64 bit mask applied to every pattern. Only these bits for each pattern will be masked against.

patnloop0 [n_cycles]

[Ctb][Moench][Mythen3] Number of cycles of loop 0.

patnloop1 [n_cycles]

[Ctb][Moench][Mythen3] Number of cycles of loop 1.

[Ctb][Moench][Mythen3] Number of cycles of loop 2.

Period between frames [Gotthard2] Uploaded to detector just before acquisition starts

periodl

[Gotthard][Jungfrau][Mythen3][Gotthard2][CTB][Moench] Period left for current frame. [Gotthard2] only in continuous mode.

port [n]

Port number of the control server on detector for detector-client tcp interface. Default is 1952. Normally unchanged.

powerchip [0, 1]

[Jungfrau][Mythen3][Gotthard2][Moench] Power the chip. Default 0. [Jungfrau] Get will return power status.Can be off if temperature event occured (temperature over temp_threshold with temp_control enabled. [Mythen3] If module not connected or wrong module, 1 will fail. By default, not powered on [Gotthard2] If module not connected or wrong module, 1 will fail. By default, powered on at server start up.

programfpga [fname.pof | fname.rbf]

[Jungfrau][Ctb][Moench] Programs FPGA from pof file. [Mythen3][Gotthard2] Programs FPGA from rbf file.

pulse [n_times] [x] [y]

[Eiger] Pulse pixel n number of times at coordinates (x, y).

Data streaming from receiver enable (eg. to GUI ot another process for further processing). 1 enables zmq data stream (creates zmq streamer threads), 0 disables (destroys streamer threads).
